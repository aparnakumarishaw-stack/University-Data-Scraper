# University & Course Data Scraper (AI/ML Intern Assignment)

## Objective
Automated scraping of Indian University data and their respective courses into a structured, relational Excel format.

## Features
- **Dynamic Scraping:** Uses Selenium to navigate university portals.
- **Relational Integrity:** Links Universities and Courses using unique `university_id` keys.
- **Data Cleaning:** Implements a cleaning pipeline for standardized text and handling missing values.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python scraper_script.py`
